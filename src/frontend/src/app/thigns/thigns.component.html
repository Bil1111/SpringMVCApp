Допомога речами

Якщо не має можливості допомогти фінансово або приїхати до притулку, то можете допомогти притулкам речами

Притулкам завжди потрібні продукти харчування:

  • Крупа (рис, гречка, ячмінна, пшенична)
  • Консерви для собак
  • Сухий та вологий корм для котів
  • М’ясні продукти (обрізки, супові набори, субпродукти)
  • Хліб, сухарі
  • Кефір, творог

Притулкам завжди потрібні такі медикаменти:

  • Хлоргексидин
  • Чемі-спрей
  • Фукорцин
  • Шприці (2-5 мл)
  • Вата
  • Бинти стерильні/нестирильні

Притулкам завжди потрібні речі:

  • Повідки
  • Ошийники
  • Намордники
  • Ковдри
  • Миючі засоби (для посуду, для миття різних поверхонь, для вікон)

Соціальна допомога:

  • Пошук господарів для тварин
  • Пошук людей для допомоги
  • Поширення інформації
